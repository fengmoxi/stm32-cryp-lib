STM32 Cryptographic Library Examples update history

 

V2.0.6 / 25- June-2013

Main Changes

•  New release for STM32 Cryptographic Library examples to support STM32F0xx, STM32F10x, STM32L1xx, STM32F2xx, STM32F37x/38x, STM32F30x/31x and STM32F4xx.

•  Expected output data are added in almost examples in order to check that the examples work correctly. Expected data are taken from cryptographic standard. Please refer to readme.txt in each example for more details about cryptographic standard used.


V2.0.5 / 31- January-2013

Main Changes

•  Internal version.

  

V2.0.4 / 02- August-2012

Main Changes

•  New release for STM32 Cryptographic Library examples to support AES 128 bits algorithm :

  ECB (Electronic Codebook Mode).

  CBC (Cipher-Block Chaining) with support for Ciphertext Stealing.

  CTR (CounTer Mode).

  CCM (Counter with CBC-MAC).

  GCM (Galois Counter Mode).

  CMAC.

  KEY WRAP.
