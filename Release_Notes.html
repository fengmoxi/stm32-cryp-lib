V2.0.6 / 25- June-2013

Main Changes

  • Update of STM32 Cryptographic Library version for STM32F10x, STM32L1xx and STM32F2xx families without changing API vs last STM32 cryptographic library.

  • New STM32 Cryptographic Libraries firmware in order to support STM32F0xx, STM32F37x/38x, STM32F30x/31x  and STM32F40x.

V2.0.0 / 01-July-2011

Main Changes

  • New algorithms and modes added vs previous version 1.0.0.
  • New STM32 Cryptographic Library version for STM32F10x, STM32L1xx and STM32F2xx families.
  • New optimized STM32 Cryptographic Library for STM32F21x using Cryptographic peripherals accelerators.
  • All algorithms supported are:
    • AES-128, AES-192, AES-256 bits. 
      • ECB (Electronic Codebook Mode)
      • CBC (Cipher-Block Chaining) with support for Ciphertext Stealing
      • CTR (CounTer Mode)
      • CCM (Counter with CBC-MAC)
      • GCM (Galois Counter Mode)
      • CMAC
      • KEY WRAP
    • DES, TripleDES. Supported Modes:
      • ECB (Electronic Codebook Mode)
      • CBC (Cipher-Block Chaining) 
    • HASH functions with HMAC support:
      • MD5 
      • SHA-1
      • SHA-224 
      • SHA-256
    • Random Engine based on DRBG-AES-128
    • RSA signature functions with PKCS#1v1.5 
    • ARC4
    • ECC (Elliptic Curve Cryptography):
      • Key Generation
      • Scalar Multiplication (the base for ECDH)
      • ECDSA  
  • Support STM32F2xx and STM32L1xx famillies.
  • Support HW algorithms for STM32F21x.
